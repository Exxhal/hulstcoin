Hulstcoin 
================================

What is Hulstcoin?

Hulstcoin is a clone of Litecoin/Funcoin 
Hulstcoin properties:

-Initial reward of 20 coins

-Block interval of 5 minutes

-Maximum supply of 350k coins

[Hulstcoin.nl](https://www.hulstcoin.nl/)
